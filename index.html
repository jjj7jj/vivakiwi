Bienvenido a VivaKiwi

Plataforma que conecta a inmigrantes y turistas hispanohablantes con proveedores de servicios confiables en Nueva Zelanda. Usa el menú para navegar por tus reservas, perfil, mensajes y configuración.

Servicios Destacados

Recepción en Aeropuerto (24/7) Transporte Privado/Local Tour por Auckland (en español) Visita a Kiwi Farm Apoyo con Trámites Básicos Ayuda con Arriendo Inicial Conexión con Iglesia o Comunidad Latina

Flujo de Uso para Clientes

  1. Entra a VivaKiwi.nz
  2. Elige idioma: 🇪🇸 Español o 🇬🇧 Inglés
  3. Crea cuenta con email o WhatsApp
  4. Selecciona servicio (ej. Recogida + Tour)
  5. Paga online (Stripe, tarjeta)
  6. Recibe confirmación y contacto del proveedor
  7. Deja valoración después del servicio

Modelo de Negocio

  • Comisión por cada servicio reservado (10-20%)
  • Publicidad local o membresías premium para proveedores

Estructura de la Plataforma

Usuarios

  • Reservan servicios fácilmente desde móvil.
  • Perfil básico: nombre, idioma, país, fecha de llegada.
  • Pago online seguro y sistema de calificaciones.

Proveedores

  • Registro con verificación de identidad.
  • Perfil con servicios, tarifas y disponibilidad.
  • Acceso a app o panel para aceptar reservas.

Admin (Tú)

  • Panel para gestionar usuarios y proveedores.
  • Control de comisiones y pagos automáticos.
  • Control de reputación y reportes.

Tecnología Recomendada

Fase 1: MVP (Producto mínimo viable)

  • Sitio web WordPress con plugins:
    • WooCommerce (pagos)
    • Bookly (reservas)
    • User Registration / WP Job Manager (registro proveedores)
  • Dominio personalizado: vivakiwi.nz o vivakiwi.com
  • Diseño adaptable a móvil (responsive)

Fase 2: Plataforma tipo app (si crece)

  • App móvil en Flutter o React Native
  • Integrar mapas (Google Maps), geolocalización y notificaciones push
  • Conectar con Stripe para pagos seguros y depósitos automáticos

Marketing y Expansión

  • Red de confianza: Usa tu iglesia latina como red inicial de usuarios y proveedores.
  • Redes sociales: Instagram y TikTok con testimonios reales.
  • Convenios con dueños de Kiwi Farms para experiencias exclusivas.
  • Blog con consejos útiles para inmigrantes → mejora SEO.
